# Vanilla JS Snake
Simple Snake game in a single HTML file. Open `index.html` or run a static server.

## Dev
- Open `index.html` directly, or
- `python -m http.server 5173` then open http://localhost:5173/

## Controls
Arrows/WASD  ·  Space = Pause  ·  R = Restart  ·  +/- = Speed
